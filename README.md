# 健身房管理系统 java1109

## 项目概述

本项目是基于Java与SpringBoot框架，结合Vue实现的前后端分离的健身房管理系统。

## 技术栈

### 系统架构
- B/S结构
- MVC模式

### 系统环境
- Windows/Mac

### 开发环境
- IDEA
- JDK1.8
- Maven
- Mysql

### 技术栈
- Java
- Mysql 8.0
- SpringBoot
- Mybatis
- Vue

## 功能模块

### 用户功能
- 用户登录

### 管理员功能
- 会员管理
- 健身房管理（包括课程管理，器材管理）
- 员工管理
- 报名表管理

### 会员功能
- 课程查询
- 在线充值
- 充值记录
- Vip管理
- 购买记录
- 我的资料
- 修改密码

## 系统角色
- 系统管理员
- 员工
- 会员

## 运行环境
- 服务器：推荐使用Java兼容的Web服务器如Tomcat
- 数据库：Mysql 8.0及以上

## 部署步骤
1. 搭建Java开发环境，确保JDK版本为1.8或以上
2. 安装并配置Mysql数据库服务
3. 使用IDEA或Eclipse导入项目源码
4. 配置项目数据库连接信息
5. 编译并运行后端服务
6. 前端项目使用Vue CLI运行
